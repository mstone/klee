# NAME

klee - symbolic LLVM interpreter

# SYNOPSIS

*klee* BITCODE_FILE

# OPTIONS

Many. See `klee --help`.

# DESCRIPTION

`klee` searches the space of executions of LLVM bitcode programs.

# EXAMPLES

None.

# SEE ALSO

`stp`(3), `s2e`(1).
