# NAME

ktest-tool - klee test-case viewer

# SYNOPSIS

*ktest-tool* KTEST_FILE

# OPTIONS

-h, \-\-help
:    show this help message and exit

\-\-trim-zeros
:    trim trailing zeros

\-\-write-ints
:    convert 4-byte sequences to integers

# DESCRIPTION

`ktest-tool` pretty-prints the contents of the specified ktest file.

# EXAMPLES

None.

# SEE ALSO

`klee`(1).
